# Peykan UI Kit

**Empty — no dedicated UI Kit file yet.** As of 2026-06-12 the 📦 Peykan Components Figma file is
empty. Note that **Peykan is a legacy product** and predates Swiss Army tokenization
(see [tokens.md](../design-system/tokens.md)), so treat any Peykan styling as not-yet-migrated.

For new Peykan work, prefer **[Swiss Army](../design-system/overview.md)** components and current
tokens, and flag legacy patterns for migration rather than reproducing them.

When a kit is built, document it here following the [Zhina](zhina-kit.md) / [TashilPay](tashilpay-kit.md) format.
