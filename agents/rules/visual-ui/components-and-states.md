# Real components & all states

- Build only from **real Swiss Army + product UI-Kit components** and **tokens** — never one-off styling.
- Design every state: default, hover/press, focus, disabled, loading, empty, error.
- If a needed component or token is missing, hand off to the Design System Manager — don't improvise it.
