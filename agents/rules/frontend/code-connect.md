# Build from the design system

- Implement UI from **real Swiss Army / UI-Kit components** and **tokens** — never hard-code colors, spacing, or type.
- Keep each component's **Code Connect** mapping in sync with the Figma source.
- Honor **RTL/Persian**: logical CSS properties (inline-start/end), Persian numerals, Jalali dates, ﷼.
