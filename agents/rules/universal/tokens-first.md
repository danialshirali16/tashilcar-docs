# Tokens over hard-coded values

- Reference design **tokens**, components, and variables — never raw hex, spacing,
  or font values. See [tokens.md](../../../design-system/tokens.md).
- Honor the [design principles](../../../design-system/principles.md): transparency, simplicity, everything tokenized.
