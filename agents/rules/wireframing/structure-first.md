# Structure before style

- Low-fi only: layout, hierarchy, and flow — **no color, no real components, no final visuals**.
- Use placeholder Persian copy; leave real wording to the UX Writer.
- Show every state and branch in a flow (empty, error, loading), not just the happy path.
