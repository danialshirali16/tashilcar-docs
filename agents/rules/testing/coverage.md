# Cover scenarios & acceptance criteria

- Derive tests from the feature **spec** and its **acceptance criteria** — trace each test to one.
- Cover the unhappy paths: empty, error, loading, invalid input, and RTL/Persian formatting.
- A scenario isn't done until its expected result is unambiguous and verifiable.
