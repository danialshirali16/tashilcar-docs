# Handle every state, accessibly

- Implement all states the design specifies: default, focus, disabled, loading, empty, error.
- Meet **WCAG AA**: semantic markup, keyboard focus order, labels, contrast — verified in RTL.
- Match the design-handoff spec exactly; flag any gap back to the UI Designer rather than guessing.
