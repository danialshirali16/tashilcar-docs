# Zamyad UI Kit

**Empty — no dedicated UI Kit file yet.** As of 2026-06-12 the 📦 Zamyad Components Figma file is
empty.

Until a Zamyad kit exists, design [Zamyad](../products/zamyad.md) screens directly from
**[Swiss Army](../design-system/overview.md)** components ([catalog](../design-system/components/_index.md))
and shared tokens. Where Zamyad needs a composite similar to an existing one, reference the
[Zhina kit](zhina-kit.md) (closest dashboard pattern) for structure — but do not assume those
components are published for Zamyad.

When the kit is built, document it here following the [Zhina](zhina-kit.md) / [TashilPay](tashilpay-kit.md) format.
